# 动手造轮子-一个简单的数据库连接池  

本工程参考方腾飞《并发编程艺术》4.4.2小节，与 https://blog.csdn.net/aloysjun/article/details/88716636，实现了一个数据库连接池。

实现连接池关键点：

- 读取配置文件，载入数据库驱动对象，创建Connection对象等
- 池化技术
- 处理并发访问，加锁、解锁
- 支持连接池数量动态扩大
- 超时机制的实现
